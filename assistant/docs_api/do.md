第一步：启动核心后端服务 (FastAPI)
你必须先让带有大模型推理、向量检索功能的服务端跑起来，并一直挂在后台。 请在你的终端（刚才敲代码的那个黑框里）执行这条命令：

bash
uvicorn app:app --host 0.0.0.0 --port 8000 --reload
注：这条命令会拉起 app.py，并在本机的 8000 端口开启一个能让别的地方调用的聊天大本营。看到终端里打印出 Application startup complete. 就说明启动成功了，千万不要关掉这个窗口。

第二步：启动交互终端 (测试对话)
此时，你需要再新开一个单独的命令行窗口。在新的窗口进入到相同的项目目录 d:\bigmodel\llmodel\assistant，然后输入：

bash
python chat_cli.py
这就如同打开了一个微信聊天框，你在这个地方打字提问，它就会向第一步启动的服务端发请求，最后把带有各种上下文知识的高质量回答“一个字一个字”地流式打印给你看了！

💡 (附赠) 如果以后更新了文档怎么办？
如果你后续向 data/docs 文件夹里丢了新的说明书 Markdown 或者文本文件，为了让模型学会这些新资料，你只需要在命令行执行一次：

bash
python ingest.py --mode rebuild
等它跑完以后，它就完全读懂并切分吸纳了你的最新知识，你就可以继续顺滑地问它了。
